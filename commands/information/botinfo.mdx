# Bot Info Command 🤖

The `/botinfo` command provides comprehensive information about Scare Bot, including statistics, uptime, and system information.

## Usage

```bash
,botinfo
```

## Description

This command displays detailed information about Scare Bot, including:

- **Bot Statistics**: Number of servers, users, and commands
- **System Information**: Uptime, latency, and memory usage
- **Bot Details**: Creation date, version, and technical specifications
- **Performance Metrics**: Real-time system performance data

## Command Details

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| None | - | No | No parameters required |

## Examples

### Basic Usage
```bash
,botinfo
```

### Expected Output
```
🤖 Scare Bot Info 🤖

Scare is serving 1,234 guilds & 45,678 users
Join the support server here

📊 Statistics
📊 commands: 486
⏰ started: 2 days ago
📅 created: January 15, 2024
🏓 latency: 45ms
📄 lines: 15,000

Memory: 128.5 MB 12%
```

## Features

### Real-Time Statistics
- **Server Count**: Current number of Discord servers
- **User Count**: Total number of users across all servers
- **Command Count**: Number of available commands (486+)

### System Performance
- **Uptime**: How long the bot has been running
- **Latency**: Response time to Discord API
- **Memory Usage**: Current memory consumption
- **CPU Usage**: System resource utilization

### Visual Elements
- **Random Emojis**: Each time you run the command, it displays different emojis
- **Colorful Embed**: Beautiful embed with bot avatar and information
- **Interactive Elements**: Links to support server and website

## Permissions

| Permission | Required | Description |
|------------|----------|-------------|
| Send Messages | Yes | Bot needs to send the response |
| Embed Links | Yes | Bot needs to send embeds |
| Send Messages | Yes | User needs to send messages |

- ✅ Ensure the bot is online and responsive
