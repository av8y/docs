# Invite Command 🔗

The `/invite` command provides beautiful invite links for Scare Bot with random emojis and engaging messages.

## Usage

```bash
,invite
```

## Description

This command generates a beautiful embed with invite links for Scare Bot, featuring:

- **Random Emojis**: Different emojis each time you run the command
- **Engaging Messages**: 25+ unique messages to make invites more appealing
- **Multiple Buttons**: Direct links to invite, support, and vote
- **Custom Design**: Light blue theme matching the bot's branding

## Command Details

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| None | - | No | No parameters required |

## Examples

### Basic Usage
```bash
,invite
```

### Expected Output
```
🚀 Invite Scare to your server! 🚀

Ready to join the ultimate Discord experience? 🚀

Click the buttons below to invite me to your server, join our support community, or vote for us!

[Invite Me] [Support Server] [Vote for Us]
```

## Features

### Random Elements
- **40+ Emojis**: Different emojis for each use (🤖, ✨, 🎉, 🚀, 💫, 🌟, 🔥, 💎, ⚡, 🎯, etc.)
- **25+ Messages**: Unique messages like:
  - "Ready to join the ultimate Discord experience? 🚀"
  - "Come be part of something amazing! ✨"
  - "Join our awesome community! 🎉"
  - "Your next adventure awaits! 💫"
  - "Ready to level up your Discord server? 🔥"

### Interactive Buttons
- **Invite Me** 🤖: Direct invite link with administrator permissions
- **Support Server** 🏠: Link to the official support Discord server
- **Vote for Us** 🔗: Link to vote for the bot on top.gg

### Visual Design
- **Light Blue Theme**: Matches the bot's color scheme (#87CEEB)
- **Bot Avatar**: Displays the bot's current avatar
- **User Footer**: Shows who requested the invite
- **Professional Layout**: Clean, modern embed design

## Permissions

| Permission | Required | Description |
|------------|----------|-------------|
| Send Messages | Yes | Bot needs to send the response |
| Embed Links | Yes | Bot needs to send embeds |
| Send Messages | Yes | User needs to send messages |

## Troubleshooting

### Command Not Working?
- ✅ Ensure the bot is online and responsive
- ✅ Check if you have permission to send messages
- ✅ Verify the bot has proper permissions in the channel

### Buttons Not Working?
- ✅ Check if the links are accessible
- ✅ Verify the bot's client ID is correct
- ✅ Ensure Discord's OAuth2 system is working

### Embed Not Displaying?
- ✅ Check if the bot has "Embed Links" permission
- ✅ Verify the channel allows embeds
- ✅ Ensure the bot can send messages in the channel
